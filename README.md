# Hugo Installation Guide

## Package Managers

```bash
# Debian/Ubuntu/Linux Mint
sudo apt update && sudo apt install hugo

# Arch Linux/Manjaro
sudo pacman -S hugo

# Fedora
sudo dnf install hugo

# openSUSE
sudo zypper install hugo

# CentOS/RHEL
sudo yum install hugo

# Alpine Linux
sudo apk add hugo

# FreeBSD
sudo pkg install hugo

# OpenBSD
sudo pkg_add hugo

# Slackware
sbopko -i hugo 
"install first sbopkg on slackware"

# Using Go (Universal)
go install github.com/gohugoio/hugo@latest

# Using Snap (Universal)
sudo snap install hugo

# Verify Installation
hugo version
```
2. Clone the repository:
```bash
git clone https://github.com/YOUR-USERNAME/hugo-bearblog.git
cd hugo-bearblog
```

3. Start the development server:
```bash
go to exampleSite/ and type "hugo server --source ./exampleSite --themesDir ../.."
```

4. Visit `http://localhost:1313` in your browser
